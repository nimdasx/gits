ubuntu desktop pin to dash

Make the AppImage executable: Right-click the AppImage file > Properties > Permissions > check "Allow executing file as program," or run chmod +x /path/to/your/AppImage in the terminal.

Create the .desktop file: Open a text editor and create a file named YourAppName.desktop in ~/.local/share/applications/.

Add content to the .desktop file:
[Desktop Entry]
Name=Your App Name
Exec=/path/to/your/AppImage %U
Icon=/path/to/your/icon.png
Type=Application
Categories=Utility;

Update the desktop database: Run update-desktop-database ~/.local/share/applications in the terminal.

Pin it: Search for your app in the Activities overview, right-click its icon, and select Add to Favorites, or drag it to the dash.
