# Cara Menyalakan Sony Xperia 1 Mark II (Xperia 1 II) Tanpa Tombol Power

Jika tombol power rusak, Anda dapat menggunakan metode berikut untuk menyalakan perangkat:

### Langkah-langkah:
1. **Tahan tombol Volume Up**.
2. **Sambungkan kabel data** ke PC atau Laptop.
3. **Pastikan lampu indikator HP menyala biru**.
4. **Jalankan perintah** `fastboot reboot` melalui terminal/command prompt.

---

### Cara Install Fastboot
Untuk menjalankan perintah di atas, Anda memerlukan **Android SDK Platform-Tools**:

1. Install **Android Studio**.
2. Buka menu SDK Manager, lalu aktifkan/install **Android SDK Platform-Tools**.
3. **Lokasi File (untuk macOS):**
   `~/Library/Android/sdk/platform-tools/fastboot`
